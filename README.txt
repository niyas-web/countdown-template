# countdown HTML Template

A minimal, clean Coming Soon page with a 2-day dynamic countdown timer.

## Features
- Fully responsive layout
- JavaScript-based live countdown
- Easy customization
- Perfect for product or portfolio launches

## How to Use
1. Download or clone this repo
2. Replace logo and background image inside `assets/`
3. Open `index.html` in any browser
4. To edit the countdown time, go to `script.js`
